## 🚀 사용 방법

### 1단계: Cloudflare 대시보드 접속
```
https://dash.cloudflare.com/
```

### 2단계: D1 데이터베이스로 이동
1. 왼쪽 메뉴 **Workers & Pages** 클릭
2. 상단 탭 **D1** 클릭  
3. **webapp-production** 선택

### 3단계: Console 탭 열기
1. 데이터베이스 상세 페이지에서 **Console** 탭 클릭
2. SQL 입력창 확인

### 4단계: SQL 파일 복사
1. 아래 경로에서 SQL 파일 열기:
   ```
   /home/user/webapp/cloudflare_console_complete.sql
   ```
2. **전체 내용 복사** (Ctrl+A → Ctrl+C 또는 Cmd+A → Cmd+C)

### 5단계: 붙여넣기 및 실행
1. Cloudflare Console SQL 입력창에 **붙여넣기** (Ctrl+V 또는 Cmd+V)
2. **Execute** 버튼 클릭
3. 실행 완료까지 대기 (약 2-3분)

### 6단계: 확인
브라우저에서 아래 URL을 열어 확인:
