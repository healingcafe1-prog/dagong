## 🔥 캐시 강제 제거 방법

### 방법 1: 브라우저 캐시 제거 (권장)

#### Android Chrome
1. 주소창에 `chrome://settings/siteData` 입력
2. 검색창에 `dagong.co.kr` 또는 `novita.ai` 입력
3. 사이트 데이터 삭제
4. 또는:
   - 설정 → 개인정보 보호 및 보안
   - 인터넷 사용 기록 삭제
   - **캐시된 이미지 및 파일** 선택
   - 기간: 전체 기간
   - 데이터 삭제

#### iOS Safari
1. 설정 → Safari
2. **기록 및 웹 사이트 데이터 지우기**
3. 또는:
   - 설정 → Safari → 고급 → 웹 사이트 데이터
   - dagong.co.kr 또는 novita.ai 검색
   - 쓸어서 삭제

#### Samsung Internet
1. 메뉴(≡) → 설정
2. 개인정보 보호 및 보안
3. 인터넷 사용 기록 삭제
4. **캐시된 이미지 및 파일** 선택
5. 삭제

### 방법 2: 시크릿 모드 (Private Browsing)
- **Chrome**: 새 시크릿 창 열기
- **Safari**: 개인 정보 보호 모드
- **Samsung Internet**: 시크릿 모드

### 방법 3: 강제 새로고침
- **Android Chrome**: 주소창 새로고침 버튼 길게 누르기
- **iOS Safari**: 주소창 새로고침 버튼 탭

## ✅ 모바일 UI 체크리스트

### 1. 하단 네비게이션 (화면 하단 고정)
- [ ] 🏠 **홈** 버튼 보이는지
- [ ] 📂 **카테고리** 버튼 보이는지
- [ ] 🔍 **검색** 버튼 보이는지
- [ ] ➕ **상품등록** 버튼 보이는지
- [ ] 👤 **마이로그인** 버튼 보이는지

### 2. 카테고리 바 (상단)
- [ ] "카테고리" 버튼 클릭 시 **상단에 카테고리 바가 나타나는지**
- [ ] 카테고리 바가 **좌우 스크롤 되는지**
- [ ] 카테고리 아이콘이 **원형으로 표시되는지**
- [ ] 홈, 한국차, 공예품, 선물, 특산물, 지역, 체험, 이벤트 **8개 카테고리** 보이는지

### 3. 상품 그리드
- [ ] 상품이 **2열로 표시되는지** (모바일)
- [ ] 상품 이미지가 **정사각형 (1:1 비율)**인지
- [ ] 상품 간격이 **적당한지** (너무 좁거나 넓지 않은지)

### 4. 반응형
- [ ] 화면을 **가로로 돌렸을 때** 레이아웃이 정상인지
- [ ] 태블릿 크기에서 **적절하게 표시되는지**

## 🐛 문제 해결

### 문제: 하단 네비게이션이 보이지 않음
**해결책**:
1. 브라우저 캐시 삭제 (위 가이드 참조)
2. 페이지 강제 새로고침
3. 시크릿 모드로 접속

### 문제: 카테고리 바가 나타나지 않음
**해결책**:
1. 하단 "카테고리" 버튼을 **클릭**했는지 확인
2. 화면 상단을 위로 스크롤했는지 확인 (숨겨져 있을 수 있음)
3. 캐시 삭제 후 재시도

### 문제: 상품이 2열로 표시되지 않음
**해결책**:
1. 모바일인지 확인 (데스크탑은 4열)
2. 캐시 삭제 후 재시도
3. 다른 브라우저로 시도

### 문제: 화면이 깨져 보임
**해결책**:
1. 브라우저 캐시 완전 삭제
2. 앱 데이터 삭제 (Android)
3. Safari 데이터 삭제 (iOS)

## 🔍 개발자 도구 확인

### Chrome DevTools (Android)
1. Chrome 주소창에 `chrome://inspect` 입력 (PC)
2. 스마트폰 USB 연결
3. 스마트폰에서 USB 디버깅 허용
4. 페이지 검사 시작

### Safari Web Inspector (iOS)
1. iPhone 설정 → Safari → 고급 → 웹 속성 활성화
2. Mac Safari → 개발 메뉴 → [iPhone 이름] → 페이지 선택

### 확인 사항:
- Console에 에러 메시지 없는지
- Network 탭에서 CSS/JS 로드 확인
- `style.css?v=숫자` 형식으로 로드되는지
- `mobile-bottom-nav` 클래스가 있는지

## 🚀 캐시 문제 완전 해결

### PC에서 강제 배포
```bash
cd /home/user/webapp

# 빌드
npm run build

# Cloudflare Pages 배포
npx wrangler pages deploy dist --project-name dagong

# 또는 프로덕션 배포
npm run deploy:prod
```

### 배포 후 확인
1. https://dagong.co.kr 접속
2. 브라우저 캐시 삭제
3. 페이지 강제 새로고침 (Ctrl+Shift+R / Cmd+Shift+R)
4. 모바일 UI 확인
