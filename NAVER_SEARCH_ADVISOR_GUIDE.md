## 🚀 네이버 서치어드바이저 등록 단계

### 1단계: 네이버 서치어드바이저 접속

```
https://searchadvisor.naver.com/
```

1. 네이버 계정으로 로그인
2. 첫 방문이면 간단한 약관 동의

---

### 2단계: 사이트 등록

1. **"사이트 관리"** 메뉴 클릭
2. **"사이트 등록"** 버튼 클릭
3. URL 입력:
   ```
   https://dagong.co.kr
   ```
4. **"확인"** 클릭

---

### 3단계: 소유권 확인 (2가지 방법)

#### 방법 1: HTML 파일 업로드 (가장 쉬움) ⭐⭐⭐⭐⭐

1. 네이버에서 제공하는 HTML 파일 다운로드
   - 파일명 예시: `naver1234567890abc.html`

2. **제게 파일 내용을 알려주시면 제가 배포해드립니다!**
   
   예시:
   ```
   naver-site-verification: naver1234567890abc.html
   ```

3. 제가 배포하면 다음 URL에서 접근 가능:
   ```
   https://dagong.co.kr/naver1234567890abc.html
   ```

4. 네이버 서치어드바이저에서 **"소유확인"** 클릭

---

#### 방법 2: 메타 태그 (대안)

1. 네이버가 제공하는 메타 태그 복사
   ```html
   <meta name="naver-site-verification" content="xxxxx" />
   ```

2. **제게 content 값을 알려주시면 제가 추가하고 배포해드립니다!**

---

### 4단계: sitemap.xml 제출 ✅

소유권 확인 완료 후:

1. **"사이트 관리"** → **"요청"** → **"사이트맵 제출"**
2. sitemap.xml URL 입력:
   ```
   https://dagong.co.kr/sitemap.xml
   ```
3. **"확인"** 클릭

✅ **sitemap.xml은 이미 준비되어 있습니다!**

---

### 5단계: RSS 제출 (선택사항)

필요 시 RSS 피드도 등록 가능합니다.

## 💡 자주 묻는 질문 (FAQ)

### Q1: sitemap.xml을 제출했는데 색인이 안 돼요
**A**: 1-3일 정도 기다려주세요. 네이버봇이 크롤링하는 데 시간이 걸립니다.

### Q2: 검색해도 사이트가 안 나와요
**A**: 신규 사이트는 2-4주 정도 걸립니다. 꾸준히 콘텐츠를 업데이트하세요.

### Q3: 특정 페이지만 색인이 안 돼요
**A**: "요청" 메뉴에서 URL 수집 요청을 개별적으로 할 수 있습니다.

### Q4: 검색 순위를 높이려면?
**A**: 
- 양질의 콘텐츠 작성
- 페이지 로딩 속도 개선
- 모바일 최적화
- 외부 링크 확보
