# 📸 Instagram 링크 추가 - 빠른 실행 가이드

**목표**: Instagram 프로필에 다공 웹사이트 링크 추가  
**소요 시간**: 1분  
**난이도**: ⭐ (매우 쉬움)

---

## 🚀 **지금 바로 실행 (1분)**

### **📱 모바일 앱 (추천)**

```
1. Instagram 앱 실행
   ↓
2. 오른쪽 하단 프로필 아이콘 탭
   ↓
3. "프로필 수정" 버튼 탭
   ↓
4. "웹사이트" 필드 탭
   ↓
5. 입력: https://dagong-bi1.pages.dev
   ↓
6. 오른쪽 상단 "완료" 탭
   ↓
7. ✅ 완료! 프로필에서 링크 확인
```

### **💻 PC 웹 브라우저**

```
1. https://www.instagram.com/korea_teacraft/ 접속
   ↓
2. 로그인
   ↓
3. "프로필 편집" 버튼 클릭
   ↓
4. "웹사이트" 필드에 입력:
   https://dagong-bi1.pages.dev
   ↓
5. 파란색 "제출" 버튼 클릭
   ↓
6. ✅ 완료! 프로필에서 링크 확인
```

## 📊 **SEO 효과**
