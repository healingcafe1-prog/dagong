## 🔧 **문제 해결**

### **"등록이 반려되었습니다" 오류**

**일반적인 사유:**
1. **사이트 접속 불가:** DNS 또는 서버 문제
2. **내용 부족:** 페이지 수가 너무 적음 (최소 5개 이상 권장)
3. **중복 등록:** 이미 등록된 사이트
4. **설명 불일치:** 사이트 설명과 실제 내용이 다름

**해결:**
- 사이트 정상 작동 확인
- 콘텐츠 보강 후 재등록
- 설명을 더 정확하게 수정

### **검색에 안 나옴**

**원인:** 색인되지 않음

**해결:**
1. robots.txt에서 `Daumoa` 허용 확인
2. 사이트맵에 모든 페이지 포함 확인
3. 2~3주 대기 (Daum 색인 속도 느림)
