## 1️⃣ Google Search Console 등록

### **Step 1: 로그인 및 속성 추가**

1. **Google Search Console 접속**
   ```
   https://search.google.com/search-console
   ```

2. **Google 계정으로 로그인**

3. **"속성 추가" 버튼 클릭**

4. **URL 입력**
   ```
   https://dagong-bi1.pages.dev
   ```
   또는
   ```
   https://dagong.co.kr  (도메인이 있을 경우)
   ```

---

### **Step 2: 소유권 확인**

Google은 여러 방법을 제공합니다. **권장: HTML 태그 방법**

#### **방법 1: HTML 태그 (권장)**

1. Google이 제공하는 메타 태그 복사
   ```html
   <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" />
   ```

2. 저에게 이 태그를 알려주시면, HTML에 추가해드리겠습니다.

3. 태그 추가 후 "확인" 버튼 클릭

#### **방법 2: HTML 파일 업로드**

1. Google이 제공하는 HTML 파일 다운로드
2. 저에게 파일 내용을 알려주시면, `/public/` 폴더에 추가해드리겠습니다.

#### **방법 3: DNS 레코드 (도메인 소유자)**

1. DNS 설정 페이지 접속
2. TXT 레코드 추가
3. Google이 제공하는 값 입력

---

### **Step 3: Sitemap 제출**

1. 좌측 메뉴에서 **"Sitemaps"** 클릭

2. **새 sitemap 추가**
   ```
   https://dagong-bi1.pages.dev/sitemap.xml
   ```

3. **"제출" 버튼** 클릭

4. **확인**: 상태가 "성공"으로 표시되면 완료

---

### **Step 4: URL 검사 (선택사항)**

1. 상단 검색바에 URL 입력
   ```
   https://dagong-bi1.pages.dev/
   ```

2. **"실시간 테스트"** 클릭

3. **"색인 생성 요청"** 클릭 (빠른 노출을 원할 경우)

---

## 2️⃣ 네이버 웹마스터 도구 등록

### **Step 1: 로그인 및 사이트 추가**

1. **네이버 웹마스터 도구 접속**
   ```
   https://searchadvisor.naver.com
   ```

2. **네이버 계정으로 로그인**

3. **"사이트 추가" 버튼** 클릭

4. **사이트 URL 입력**
   ```
   https://dagong-bi1.pages.dev
   ```

---

### **Step 2: 소유권 확인**

✅ **이미 완료됨!** 네이버 인증 태그가 HTML에 삽입되어 있습니다.

```html
<meta name="naver-site-verification" content="51a0658e89bc8816eeb448bf53b1862b86609662"/>
```

1. **"HTML 태그" 방법** 선택

2. **"소유확인" 버튼** 클릭

3. 자동으로 확인됩니다 ✅

---

### **Step 3: Sitemap 제출**

1. 좌측 메뉴에서 **"요청" > "사이트맵 제출"** 클릭

2. **사이트맵 URL 입력**
   ```
   https://dagong-bi1.pages.dev/sitemap.xml
   ```

3. **"확인" 버튼** 클릭

---

### **Step 4: RSS 제출 (선택사항)**

1. **"요청" > "RSS 제출"** 클릭

2. RSS 피드가 있다면 입력
   ```
   https://dagong-bi1.pages.dev/feed.xml
   ```
   (현재는 없음, 향후 추가 가능)

---

### **Step 5: 수집 요청**

1. **"요청" > "수집 요청"** 클릭

2. **주요 페이지 URL 입력**
   ```
   https://dagong-bi1.pages.dev/
   https://dagong-bi1.pages.dev/products?type=tea
   https://dagong-bi1.pages.dev/products?type=craft
   https://dagong-bi1.pages.dev/events
   ```

3. 하루 최대 10개까지 수집 요청 가능

## ❓ 자주 묻는 질문

### Q1: 등록했는데 검색이 안 돼요
**A**: 2~7일 정도 기다려주세요. Sitemap이 처리되는 시간이 필요합니다.

### Q2: 특정 페이지가 검색 안 돼요
**A**: 해당 페이지를 직접 수집 요청하세요. (URL 검사/수집 요청)

### Q3: 검색 순위가 낮아요
**A**: SEO 최적화가 필요합니다. 키워드, 콘텐츠, 백링크 등을 개선해야 합니다.

### Q4: 네이버 쇼핑 검색에도 나오나요?
**A**: 네이버 쇼핑은 별도 등록이 필요합니다. (스마트스토어 입점)
