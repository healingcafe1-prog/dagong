# Cloudflare Console 복사-붙여넣기 가이드

## 📅 생성일: 2026-02-19

## 🎯 목적
Cloudflare D1 Console에서 바로 복사-붙여넣기 가능한 SQL 파일 사용 가이드

## 📁 파일 위치
```
/home/user/webapp/cloudflare_console_ready.sql
```

**파일 크기**: 228줄  
**인코딩**: UTF-8  
**형식**: 표준 SQLite SQL

## 🚀 사용 방법

### 1단계: Cloudflare 대시보드 접속
```
https://dash.cloudflare.com/
```

### 2단계: D1 데이터베이스로 이동
1. 왼쪽 메뉴 **Workers & Pages** 클릭
2. 상단 탭 **D1** 클릭  
3. **webapp-production** 선택

### 3단계: Console 탭 열기
1. 데이터베이스 상세 페이지에서 **Console** 탭 클릭
2. SQL 입력창 확인

### 4단계: SQL 파일 복사
1. 아래 경로에서 SQL 파일 열기:
   ```
   /home/user/webapp/cloudflare_console_ready.sql
   ```
2. **전체 내용 복사** (Ctrl+A → Ctrl+C 또는 Cmd+A → Cmd+C)

### 5단계: 붙여넣기 및 실행
1. Cloudflare Console SQL 입력창에 **붙여넣기** (Ctrl+V 또는 Cmd+V)
2. **Execute** 버튼 클릭
3. 실행 완료까지 대기 (약 1-2분)

### 6단계: 확인
브라우저에서 아래 URL을 열어 확인:

**제품 확인 (50개 예상):**
```
https://dagong-bi1.pages.dev/api/products
```

**이벤트 확인 (27개 예상):**
```
https://dagong-bi1.pages.dev/api/events
```

**체험 프로그램 확인 (12개 예상):**
```
https://dagong-bi1.pages.dev/api/experiences
```

## 🔧 문제 해결

### SQL 실행 오류
- **문법 오류**: Cloudflare Console 에러 메시지 확인
- **타임아웃**: 네트워크 상태 확인 후 재시도
- **권한 오류**: 계정 관리자에게 문의

### 데이터가 보이지 않는 경우
1. 브라우저 캐시 삭제 (Ctrl+Shift+R 또는 Cmd+Shift+R)
2. 5분 후 다시 확인
3. API 직접 확인:
   ```bash
   curl https://dagong-bi1.pages.dev/api/products
   ```

### 일부 데이터만 삽입된 경우
1. Cloudflare Console에서 에러 로그 확인
2. 문제가 있는 라인 수정
3. 다시 전체 SQL 실행
